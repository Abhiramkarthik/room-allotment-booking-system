# Room Allotment and Booking System (C++)

# Overview
A CLI-based Room Allotment and Booking System developed using C++ and advanced Data Structures.
The system allows room scheduling, host assignment, time conflict management, and chair allocation.

# Features
- Add rooms with capacity
- Book rooms with time scheduling
- Prevent overlapping bookings
- Chair availability check
- View room booking status
